NovelAI Image Generation

Usage guide for the AstrBot NovelAI plugin. Chinese is shown by default; switch to English when needed.

1. 配置 token

在插件设置里填写 auth_token，只填 token 本体，不要带 Bearer。

2. 检查 endpoint

推荐保持 https://image.novelai.net。

3. 发送指令

推荐 /nai draw，也支持 /novelai、imagine 和 约稿。

常用命令

/nai help
/nai draw 1girl, cat ears, blue eyes
/nai 1girl, cat ears, blue eyes
/novelai 1girl, white dress, garden
imagine 1girl, cinematic lighting
约稿 1girl, cat ears, blue eyes

参数示例

/nai draw 1girl -r square -m nai-v3 -s k_euler_a -C native
/nai draw 1girl --width 1024 --height 1024 -t 32 -c 5.5 -x 123
/nai draw 1girl -u "bad hands, extra fingers"
/nai draw 一个女孩，猫耳，蓝眼睛
/nai draw 一个女孩，猫耳 -T
/nai draw 1girl -i 2 -b 3 -o verbose

改图和增强

消息里带图片时，插件会自动走 img2img。-N 控制改图幅度，-n 控制噪声。

/nai draw 1girl, new clothes -N 0.7 -n 0.2
/nai img 1girl, different hairstyle
/nai enhance masterpiece
/nai draw masterpiece -e

批量和输出模式

/nai draw 1girl -i 3
/nai draw 1girl -b 4
/nai draw 1girl -i 2 -b 3 -o verbose

-o minimal 只发图；-o default 发 seed 和图；-o verbose 发完整参数和图。

自动翻译

开启 auto_translate_prompt 后，包含中文的 prompt 会先交给 AstrBot 当前接入的 LLM 翻译成英文标签。如果没生效，请把平台日志切到 debug，查看 NovelAI auto-translate 开头的日志。

Translate the following NovelAI image prompt tags into concise English tags. Keep existing English tags, weights, brackets, braces, commas, and tag order. Return only the translated prompt, with no explanation:
{prompt}

参数表

-r尺寸：portrait、landscape、square 或 1024x1024。
-m模型：nai-v3、nai-v4-full 等。
-s采样器：k_euler_a 会自动转成 k_euler_ancestral。
-u负面提示词，等同 --negative 或 --undesired。
-O不自动追加默认提示词。
-T本次请求跳过自动翻译。
-N改图幅度 strength。
-n改图噪声 noise。
-e增强模式。
-i生成轮数 iterations。
-b每轮张数 batch。
-o输出模式：minimal、default、verbose。

当前支持 NovelAI 官方 token、text2img、img2img、enhance、批量生成和自动翻译。

1. Configure token

Set auth_token in plugin settings. Paste only the token value, without Bearer.

2. Check endpoint

Recommended endpoint: https://image.novelai.net.

3. Send commands

Use /nai draw, /novelai, imagine, or 约稿.

Commands

/nai help
/nai draw 1girl, cat ears, blue eyes
/nai 1girl, cat ears, blue eyes
/novelai 1girl, white dress, garden
imagine 1girl, cinematic lighting

Examples

/nai draw 1girl -r square -m nai-v3 -s k_euler_a -C native
/nai draw 1girl --width 1024 --height 1024 -t 32 -c 5.5 -x 123
/nai draw 1girl -u "bad hands, extra fingers"
/nai draw 1girl -i 2 -b 3 -o verbose

Image-to-Image and Enhance

/nai draw 1girl, new clothes -N 0.7 -n 0.2
/nai img 1girl, different hairstyle
/nai enhance masterpiece
/nai draw masterpiece -e

Batch and Output

/nai draw 1girl -i 3
/nai draw 1girl -b 4
/nai draw 1girl -i 2 -b 3 -o verbose

minimal sends image only. default sends seed and image. verbose sends full parameters and image.

Auto Translation

Enable auto_translate_prompt to translate prompts containing Chinese characters through the configured AstrBot LLM provider. If it does not work, enable debug logging and look for logs beginning with NovelAI auto-translate.

Translate the following NovelAI image prompt tags into concise English tags. Keep existing English tags, weights, brackets, braces, commas, and tag order. Return only the translated prompt, with no explanation:
{prompt}

Options

-rResolution: portrait, landscape, square, or 1024x1024.
-mModel: nai-v3, nai-v4-full, etc.
-sSampler. k_euler_a is converted to k_euler_ancestral.
-uNegative prompt.
-ODo not append default prompts.
-TSkip auto translation once.
-Nimg2img strength.
-nimg2img noise.
-eEnhance mode.
-iIterations.
-bBatch size per iteration.
-oOutput mode: minimal, default, or verbose.

Supported now: official NovelAI token backend, text2img, img2img, enhance, batch generation, and auto translation.
