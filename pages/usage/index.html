NovelAI 图片生成

AstrBot 插件使用说明。配置 auth token 后，在聊天里发送指令即可调用 NovelAI 官方图片接口。

1. 配置 token

在插件设置里填写 auth_token。只填 token 本体，不要带 Bearer。

2. 发送指令

推荐使用 /nai draw，也支持 /novelai、imagine 和 约稿。

3. 等待回图

插件会提取 NovelAI 返回 zip 中的第一张图片，并发送回当前会话。

常用命令

/nai help
/nai draw 1girl, cat ears, blue eyes
/nai 1girl, cat ears, blue eyes
/novelai 1girl, white dress, garden
imagine 1girl, cinematic lighting
约稿 1girl, cat ears, blue eyes

参数示例

/nai draw 1girl -r square -m nai-v3 -s k_euler_a -C native
/nai draw 1girl --width 1024 --height 1024 -t 32 -c 5.5 -x 123
/nai draw 1girl -u "bad hands, extra fingers"
/nai draw 1girl -O
/nai draw 1girl -S -d -D
/nai draw 一个女孩，猫耳，蓝眼睛
/nai draw 一个女孩，猫耳 -T
/nai draw 1girl -i 2 -b 3 -o verbose

改图和增强

消息里带图片时，插件会自动走 img2img。用 -N 控制改图幅度，用 -n 控制噪声。

/nai draw 1girl, new clothes -N 0.7 -n 0.2
/nai img 1girl, different hairstyle

增强模式参考 koishijs/novelai-bot：需要发送一张宽高之和为 1280 的 NovelAI 生成图，输出会放大到 1.5 倍。

/nai enhance masterpiece
/nai draw masterpiece -e

批量和输出模式

-i 是生成轮数，-b 是每轮张数。单次命令总张数会受设置里的 max_total_images 限制。

/nai draw 1girl -i 3
/nai draw 1girl -b 4
/nai draw 1girl -i 2 -b 3 -o verbose

-o minimal 只发图；-o default 发 seed 和图；-o verbose 发完整参数和图。

参数表

-rportrait、landscape、square 或 1024x1024。
-m模型或别名，例如 nai-v3、nai-v4-full。
-s采样器，例如 k_euler_a、k_euler、k_dpmpp_2m。
-xseed，范围为 0 到 4294967295。
-tsteps，本插件限制为 1 到 50。
-cscale，提示词引导强度。
-u负面提示词，等同 --negative 或 --undesired。
-O覆盖模式，不自动追加默认提示词。
-T本次请求跳过 LLM 自动翻译。
-N改图幅度 strength，越大改动越明显。
-n改图噪声 noise。
-e增强模式，等同 /nai enhance。
-i生成轮数 iterations。
-b每轮张数 batch。
-o输出模式：minimal、default、verbose。

自动翻译

开启 auto_translate_prompt 后，包含中文的 prompt 会先交给 AstrBot 当前接入的 LLM 翻译成英文标签，再发送给 NovelAI。

Translate the following NovelAI image prompt tags into concise English tags. Keep existing English tags, weights, brackets, braces, commas, and tag order. Return only the translated prompt, with no explanation:
{prompt}

这个默认模板可以修改；只要保留 {prompt}，插件就能把用户输入插入到模板里。

和 koishijs/novelai-bot 的关系

本插件不是直接依赖 Koishi 插件，而是参考并移植了它的成熟行为：模型别名、方向尺寸、短参数、默认 prompt、采样器转换、NAI v3/v4 payload、zip 图片解析、img2img 和 enhance。

当前范围

当前支持 NovelAI 官方 token 后端、text2img、img2img、enhance 和自动翻译。暂不包含批量生成、自动撤回、账号密码登录和第三方后端。
